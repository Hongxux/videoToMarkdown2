# V6.9.4 动静判断逻辑优化 - 实现总结

> 基于 `动静判断的逻辑优化v2有效动态.md` 和 `动静判断的逻辑优化v3平移类动画的识别.md` 的第一性原理，实现了以下核心优化。

## 一、核心优化点

### 1. 边缘差分累积检测优化

**问题**：原逻辑使用绝对方差过滤噪点，但实际平移动画可能不是匀速的，方差会很高（如732698），导致合法动画被误判为噪点。

**解决方案**：使用**有效帧比例**替代绝对方差

```python
# 有效帧 = 边缘变化超过最小阈值的帧
min_diff_per_frame = 10
effective_frames = sum(1 for d in edge_diffs if d > min_diff_per_frame)
effective_ratio = effective_frames / len(edge_diffs)

# 判定: 累积能量高 + 有效帧比例≥30%
is_smooth_animation = (
    cumulative_energy >= TH_EDGE_DIFF_CUMULATIVE and
    effective_ratio >= 0.3
)
```

### 2. 重心位移检测优化

**问题**：同上，绝对方差阈值(100)过于严格。

**解决方案**：同样使用有效帧比例

```python
min_disp_per_frame = 2.0  # 每帧最小位移
effective_frames = sum(1 for d in displacements if d > min_disp_per_frame)
effective_ratio = effective_frames / len(displacements)

is_smooth_animation = avg_displacement > 5.0 and effective_ratio >= 0.3
```

### 3. 帧级别动画检测（非整段标记）

**问题**：原逻辑检测到平滑动画后，将整个区间标记为ACTION（导致P002显示97%动作），但实际动画只发生在部分时间段。

**解决方案**：逐帧检测，只标记有实际动画的帧

```python
for idx, (t, frame) in enumerate(frames):
    if current_state[1] == FrameState.STABLE:
        # 检测边缘变化
        if prev_edges is not None:
            edge_diff = cv2.absdiff(edges, prev_edges)
            diff_energy = np.sum(edge_diff > 0)
            if diff_energy > 500:  # 单帧阈值
                is_frame_animated = True
        
        # 检测重心位移
        if prev_centroid and centroid:
            disp = np.sqrt(dx**2 + dy**2)
            if disp > 3.0:  # 单帧阈值
                is_frame_animated = True
        
        if is_frame_animated:
            new_states.append((t, FrameState.ACTION, 0.5))
        else:
            new_states.append(current_state)
```

### 4. 快速冗余检测修复

**问题**：边缘占比阈值(1%)对PPT类视频过于严格，正常画面(边缘占比0.4%-0.7%)被误判为冗余。

**解决方案**：移除边缘占比检测，仅保留亮度方差检测（纯黑/纯白屏）

```python
def _quick_redundancy_check(self, frame, roi):
    # 仅检查亮度方差（纯黑/纯白）
    variance = np.var(gray)
    if variance <= RED_LOW_VAR_THRESH:  # 30
        return RedundancyType.RED_BLANK
    
    # 不再检查边缘占比 - PPT类视频边缘本来就少
    return None
```

## 四、ActionUnit增强

```python
@dataclass
class ActionUnit:
    start_sec: float
    end_sec: float
    avg_diff_ratio: float
    action_type: str = "unknown"  # knowledge/transition/noise
    ssim_drop: float = 0.0
    is_effective: bool = True
    
    def classify(self) -> str:
        """动态类型分类 (第一性原理)"""
        duration_s = self.duration_ms / 1000
        if duration_s >= 1.5 and self.ssim_drop < 0.2:
            return "knowledge"    # 知识生产型
        elif duration_s < 1.5 and self.ssim_drop > 0.5:
            return "transition"   # 容器切换型
        elif duration_s < 0.3:
            return "noise"        # 无效变动
        else:
            return "mixed"
```

## 五、V6.9.5 有效动态过滤

### ssim_drop 计算

```python
# 计算真正的SSIM跌幅 = 首帧SSIM - 末帧SSIM
first_ssim = current_ssims[0]
last_ssim = current_ssims[-1]
ssim_drop = max(0.0, first_ssim - last_ssim)

# 如果区间内有极低SSIM(<0.5)，说明场景突变
min_ssim = min(current_ssims)
if min_ssim < 0.5:
    ssim_drop = 1.0 - min_ssim  # 大跌幅
```

### 分类优先级 (V6.9.5)

```python
def classify(self):
    # 1. SSIM跌幅>50% → 容器切换型 (场景突变,不论时长)
    if self.ssim_drop > 0.5:
        return "transition"
    
    # 2. SSIM跌幅<20% + 时长≥1.5s → 知识生产型
    if duration >= 1.5 and self.ssim_drop < 0.2:
        return "knowledge"
    
    # 3. 时长<0.3s → 噪点
    if duration < 0.3:
        return "noise"
    
    return "mixed"
```
