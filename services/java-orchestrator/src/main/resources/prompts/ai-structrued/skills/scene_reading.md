# Skill: 读书笔记 / 观点评论场景（scene_reading）

当 section 涉及观点评论、思想梳理、多维度分析等发散思考型内容时，适用本 skill。

---

## 精修规则

### 1. 统摄性维度归纳
- 不要机械地罗列观点（影响1、影响2、影响3），必须发掘散乱观点背后的**统摄性维度**
- 将多个碎片化观点归纳进高层次分类标签之下
- 例如：将"影响1: 工资变化"、"影响2: 就业率变化" 归纳为 `**经济影响（劳动力市场视角）**`

### 2. 分类标签的抽象层级提升
- 分类标签不能停留在表层重复（如 `**优点**：`），必须提升到具有**解释力**的抽象层级
- 标签应回答"为什么要从这个角度看？"
- 例如：不要用 `**长期影响**`，而应用 `**长期影响（重塑行业壁垒）**` — 标签本身即传递核心论断

### 3. 金句/核心论断提炼
- 对于原文中极其精辟的核心论断或反常识观点，使用加粗或 `== ==` 高亮标出
- 金句应保持原文措辞，不要改写

### 正向示例
**❌ 精修前（机械罗列）：**
```markdown
- **影响1**：改变了消费者的购买习惯
- **影响2**：重塑了零售行业的竞争格局
- **影响3**：推动了物流基础设施的升级
```

**✅ 精修后（统摄性维度 + 标签升级）：**
```markdown
- **电商革命的涟漪式影响（按受影响主体分层）**：
    - **消费端（行为范式迁移）**：从"逛街购物"到"指尖决策"，消费者的选择方式和决策路径被彻底重构。
    - **产业端（竞争壁垒重塑）**：传统零售的"位置优势"让位于数据驱动的"流量优势"，行业格局发生结构性翻转。
    - **基础设施端（底层能力倒逼升级）**：海量订单倒逼物流网络从区域分散走向全国一体化智能调度。
```
