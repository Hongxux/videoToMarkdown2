## 深度
1. 【本质剥离】：不要停留在算法描述。其本质是贝叶斯推理在序列数据上的应用：模型基于先验知识（训练数据）和当前证据（上下文），更新对下一个 token 的信念分布，这是一种信息压缩与预测的熵减过程。
2. 【第一性原理】：这触及了热力学第二定律在信息处理中的体现：通过概率生成，模型在不确定性（高熵）的语料库中，施加约束（提示）以导向低熵、高信息量的输出，是智能作为负熵流的具象化。
