## 语境化解释
1. 【概念解构（破除陌生感）】：在此语境下，'Claude 对 XML 友好'并非泛泛而谈的格式兼容，而是特指 Claude 模型在训练数据中大量接触 XML 结构，导致其内部推理机制对 XML 标签、嵌套和语义分割有原生亲和力，从而在提示词中嵌入 XML 能更精准地激活其底层模式匹配能力。
2. 【点石成金（回应好奇心）】：作者用此术语，是因为它直接呼应前文'投其所好'的调优原理，将抽象的训练数据未知困境，转化为可操作的确定性策略——利用模型在训练中形成的格式偏好，作为对抗黑盒的'已知锚点'，实现高效调优。

## 深度
1. 【本质剥离】：这触及模型训练的本质：大语言模型通过海量数据学习统计模式，XML 友好性源于训练语料中 XML 文档的高频出现，使得模型在 token 预测时对 XML 结构有更低的困惑度（perplexity），从而提升输出质量和稳定性。
2. 【第一性原理】：从信息论和认知科学看，这体现了'格式即结构，结构即认知'的底层法则——XML 的层次化标签为模型提供了明确的语义边界，减少了歧义，类似于人类大脑对结构化信息的处理效率更高。

## 广度
1. 【跨界同构】：跳出 AI，这与人类语言学习中的'母语效应'同构——就像说英语者更易掌握拉丁字母系统，Claude 对 XML 的友好性是其'数据母语'的体现，在软件工程中类似 IDE 对特定编程语言的语法高亮优化，提升开发效率。
2. 【高维演进/极端延伸】：在更广视野，这映射到生态系统中物种对特定环境的适应——如沙漠植物进化出节水机制，模型格式偏好是其在数据环境中'进化'出的生存策略，可延伸至商业中产品设计针对用户习惯的优化。
