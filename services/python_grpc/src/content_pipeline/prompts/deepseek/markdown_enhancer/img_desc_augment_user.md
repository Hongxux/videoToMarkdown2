## 图片证据（按时间/句子对齐）

请基于证据输出“增量补丁 JSON”（不要输出完整重写文本）。

输出规则：
1. 顶层键使用 `p`（patches）。
2. 每个补丁对象使用短键：
   - `m`: 模式，`r`=replace，`a`=add
   - `o`: 原片段（replace 必填）
   - `n`: 新片段/新增片段（必填）
   - `l`: 左上下文（可选，建议提供）
   - `r`: 右上下文（可选，建议提供）
   - `p`: add 的插入方向（可选，`before/after`，默认 `after`）
3. 无可补全内容时，输出 `{{"p":[]}}`。

仅输出 JSON，不要输出解释文本，不要输出代码块。
