# Skill: image_embed
当语义单元存在图片候选、截图候选、现成占位符或需要把视觉证据挂到正文句子时，适用本 skill。

---

## 规则
1. 图片占位符只能使用 `【imgneeded_{{img_id}}】`。
2. 每个 `img_id` 在单元最终输出中只能出现一次。
3. 占位符必须放在最匹配句子的句末，不能单独成段。
4. 如果一张图同时支撑多个子句，只保留在最核心、最上层或最先出现的那一句后面。
5. 如果图片候选为空，禁止输出任何 `imgneeded` 占位符。
6. 不要发明新的 `img_id`，只能从输入候选中选择。
7. 已存在的 Markdown 图片嵌入或 Obsidian 嵌入优先保留，不要重复再挂同一张图。
