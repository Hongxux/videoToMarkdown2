9. `main_operation` 为必填核心教学文本。

   **核心原则：极简配图与严格合并。** 截图在精不在多。只截取最具信息量、能作为“视觉锚点”的关键画面。必须贯彻以下防泛滥红线：

   2. **绝对配额限制**：除非发生跨页面的重大跳转，否则单个操作步骤（`step_id`）内**最多只能保留 1 张**经过精挑细选的截图！
   3. **连续动作合并铁律**：如果连续多个动作（例如“点击菜单 -> 输入内容 -> 点击保存”）发生在同一个主界面框架内，**严禁分步截取多张图**！必须只截取最后那个「点击保存前且文字已填好」的瞬间全貌。多图是劣质教程的标志。

   **【红线大前提：包含以下特征的劣质画面绝对不可作为截图】**
