# 模块1：视频内容初步整理

## 任务说明
分析原始字幕，识别创作者想要强调的内容，并纠正明显的字幕识别错误。

---

## 识别重点的依据

### 1. 语气词和强调表达
**识别方法**（灵活判断，不要机械）：
- 明确的语气词："这个很重要"、"注意"、"关键是"、"核心"、"本质"
- 强调句式："必须"、"一定"、"千万"、"务必"
- 价值判断："最好的"、"核心的"、"本质的"、"关键的"
- 反复解释同一个观点（即使没有重复相同词汇）

**AI自主判断原则**：
- 不要只看关键词，要理解创作者想强调什么
- 如果创作者花了很多时间解释某个点，即使没有明确语气词，也应标记
- 如果不确定，倾向于多标记而不是少标记

**标记**：【重点】

**示例**：
```
输入：这个很重要啊，消息持久化能确保消息不丢失
输出：【重点】消息持久化能确保消息不丢失。

输入：我必须强调一下，这里的异步模型是整个系统的基础
输出：【重点】这里的异步模型是整个系统的基础。

输入：（创作者花了3句话反复解释同一个概念）
输出：【重点】[该概念的核心表述]
```

---

### 2. 核心概念（专业术语）

**判断标准**：
1. **必须是专业术语**：该领域特有的术语或有明确定义的概念
2. **重复次数灵活判断**：
   - 如果是非常核心的专业术语，重复2次也可以标记
   - 如果是一般专业术语，需要重复3次以上
   - 如果是普通词汇，即使重复10次也不标记

**专业术语的例子**：
- ✅ 领域特有术语："唯物主义"、"辩证法"、"形而上学"、"消息队列"、"时间解耦"、"持久化"
- ❌ 动词："了解"、"知道"、"理解"、"认识"
- ❌ 泛泛的词："系统"、"方法"、"问题"、"内容"

**标记**：【核心概念：XXX】（重复X次）

**示例**：
```
✅ "消息持久化"重复3次 → 【核心概念：消息持久化】（重复3次）
✅ "唯物主义"重复2次但是核心术语 → 【核心概念：唯物主义】（重复2次）
❌ "了解"重复5次 → 不标记（不是专业术语）
❌ "系统"重复4次 → 不标记（太泛泛）
```

---

### 3. 停顿

**识别方法**：
- 长时间停顿（超过2秒）
- 可能是思考或强调

**标记**：【停顿】（X秒）

**示例**：
```
输入：[00:01:30] （停顿3秒）
输出：[00:01:30] 【停顿】（3秒）
```

---

## 处理规则

### 1. 保留原意
- 不改变创作者的表达
- 不添加或删除实质性内容

### 2. 合并句子
- 使其连贯易读
- 删除不必要的重复

### 3. 删除填充词
删除以下无意义词汇：
- "嗯"、"啊"、"呃"
- "那个"、"这个"、"就是说"
- "然后"、"所以说"（如果不影响逻辑）

### 4. 保留时间戳
- 便于后续定位
- 格式：[HH:MM:SS]

### 5. **字幕纠错（新增）**

**纠错原则**：
- 只纠正明显的同音字错误
- 根据上下文语境判断
- 不确定时保留原文
- 不纠正语法错误或标点错误

**常见错误示例**：
- "维新者" → "唯心者"
- "行而上学" → "形而上学"
- "辩正" → "辩证"
- "意园论" → "一元论"

**纠错标注方式**：
- 在正文中直接使用纠正后的词
- 在文末统一列出所有纠错

---

## 输出格式

### 正文部分
```
[时间戳] 整理后的文本
【标记】说明
```

### 文末纠错部分
```
---

## 字幕纠错记录

1. [时间戳] "原文" → "纠正" 
   判断依据：[说明为什么这样纠正]

2. [时间戳] "原文" → "纠正"
   判断依据：[说明]
```

---

## 完整示例

### 输入
```
[00:01:20] 这个...嗯...这个很重要啊，就是说，消息持久化，消息持久化它能确保，
[00:01:25] 确保消息不丢失，这个是关键，关键就在于它写入了磁盘。
[00:01:30] （停顿3秒）
[00:01:33] 所以说，持久化，持久化是整个系统的基础。
[00:02:10] 接下来讲维新者和唯物主义的区别
```

### 输出
```
[00:01:20] 【重点】消息持久化能确保消息不丢失。
【核心概念：消息持久化】（重复3次）
关键在于它写入了磁盘。

[00:01:30] 【停顿】（3秒）

[00:01:33] 持久化是整个系统的基础。
【核心概念：持久化】

[00:02:10] 接下来讲唯心者和唯物主义的区别
【核心概念：唯心者】
【核心概念：唯物主义】

---

## 字幕纠错记录

1. [00:02:10] "维新者" → "唯心者"
   判断依据：根据上下文"唯物主义"，此处应为哲学术语"唯心者"而非历史术语"维新者"
```

---

## 质量检查

在输出前确保：
- [ ] 所有明显的重点都已标记（不要遗漏）
- [ ] 核心概念都是专业术语（不是普通词汇）
- [ ] 填充词已删除
- [ ] 句子连贯易读
- [ ] 时间戳保留
- [ ] 原意未改变
- [ ] 明显的同音字错误已纠正
- [ ] 纠错记录完整，包含判断依据
