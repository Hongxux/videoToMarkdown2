# 模块2：视频内容深度分析

## 任务说明
基于模块1的输出（整理后文本 + 重点标记），提取创作者的核心洞察、逻辑链和思维模式。

---

## 输入
模块1的输出，包含：
- 整理后的文本
- 【重点】标记
- 【核心概念】标记

---

## 你的任务

### 1. 提取核心洞察

#### 主要观点
创作者的核心主张是什么？

**识别方法**：
- 从【重点】标记中提取
- 从【核心概念】中归纳
- 关注创作者反复强调的内容

**输出格式**：
```
主要观点：[一句话概括创作者的核心主张]
```

---

#### 关键干货
创作者提供的具体方法/技巧/知识点

**要求**：
- 列出3-5个最重要的干货
- 每个干货要具体、可操作
- 避免空泛的总结

**输出格式**：
```
关键干货：
1. **[干货1]**：[具体说明，包含操作步骤或关键要点]
2. **[干货2]**：[具体说明]
3. **[干货3]**：[具体说明]
```

---

#### 核心概念定义
【核心概念】的准确定义

**要求**：
- 创作者如何定义这些概念？
- 与常见理解有何不同？
- 为什么这个定义很重要？

**输出格式**：
```
核心概念：
- **[概念1]**：[创作者的定义] + [与常见理解的区别]
- **[概念2]**：[创作者的定义] + [为什么重要]
```

---

### 2. 重构逻辑链

#### 判断依据
创作者基于什么做出判断？

**识别方法**：
- 寻找"因为"、"所以"、"基于"等逻辑词
- 识别前提假设
- 找出支撑观点的事实/数据/经验

**输出格式**：
```
判断依据：
[创作者基于什么做出判断]

前提假设：
- [假设1]：[具体说明]
- [假设2]：[具体说明]

支撑事实：
- [事实1]：[具体内容]
- [事实2]：[具体内容]
```

---

#### 证据链
创作者提供了哪些证据？证据如何支撑观点？

**要求**：
- 明确 证据 → 推理 → 结论 的链条
- 展示证据之间的关系
- 指出推理的关键步骤

**输出格式**：
```
证据链：

证据1 → 结论1
- **证据**：[具体证据]
- **推理**：[如何从证据推导出结论]
- **结论**：[得出的结论]

证据2 → 结论2
- **证据**：[具体证据]
- **推理**：[如何推理]
- **结论**：[得出的结论]

证据关系：
[证据1和证据2如何相互支撑或递进]
```

---

#### 断言
创作者的明确结论是什么？

**要求**：
- 提取创作者的明确主张
- 说明适用范围
- 指出限制条件

**输出格式**：
```
断言：
[创作者的明确结论]

适用范围：
[在什么情况下这个结论成立]

限制条件：
[什么情况下这个结论不成立或需要调整]
```

---

### 3. 分析思维模式

#### 世界观
创作者如何看待这个领域/问题？

**识别方法**：
- 创作者认为什么是本质？
- 创作者认为什么是次要的？
- 创作者的价值判断标准是什么？

**输出格式**：
```
世界观：
[创作者如何看待这个领域/问题]

认为本质是：[...]
认为次要的是：[...]
价值判断标准：[...]
```

---

#### 价值观
创作者认为什么是重要的？

**要求**：
- 识别创作者的优先级排序
- 理解创作者的权衡标准
- 找出创作者的价值取向

**输出格式**：
```
价值观：
[创作者认为什么是重要的]

优先级排序：
1. [最重要]：[为什么]
2. [次重要]：[为什么]
3. [再次重要]：[为什么]

权衡标准：
[在X和Y之间，创作者如何选择]
```

---

#### 关联方式
创作者如何将概念联系起来？

**识别方法**：
- 使用了哪些类比/隐喻？
- 建立了哪些因果关系？
- 如何将抽象概念具象化？

**输出格式**：
```
关联方式：
[创作者如何将概念联系起来]

使用的类比/隐喻：
- [类比1]：[用X比喻Y，说明...]
- [类比2]：[用A比喻B，说明...]

建立的因果关系：
- [原因] → [结果]：[解释]
- [原因] → [结果]：[解释]

具象化方法：
[如何将抽象概念变得具体可感]
```

## 五、视觉辅助素材识别

### 任务说明
识别视频中可能存在的视觉演示内容，为后续的关键帧提取提供指导。

### 识别标准

**什么时候需要视觉辅助？**
- 讲解复杂概念时（如：流程图、架构图、对比表格）
- 展示数据/统计时（如：图表、数据可视化）
- 演示效果时（如：前后对比、操作演示）
- 使用PPT/板书时（如：知识点总结、公式推导）

**不需要视觉辅助的情况**：
- 纯口述内容，无视觉演示
- 只是讲话的画面，无实质性视觉内容

### 输出格式

```markdown
## 视觉辅助素材清单

### 素材1：[素材名称]
- **时间范围**：[开始时间] - [结束时间]（如：00:02:45 - 00:03:15）
- **类型**：[PPT/板书/图表/流程图/对比图/数据展示/效果演示/视频片段]
- **内容说明**：[这个视觉内容展示了什么？为什么重要？]
- **插入位置**：[应该插入到分析文档的哪个部分]（如：核心洞察 → 主要观点之后）
- **提取方式**：[关键帧/视频片段]
- **帧数建议**：[如果是关键帧，建议提取几张]（如：2-3张，因为PPT逐步展示）

### 素材2：[素材名称]
...
```

### 识别要点

1. **时间范围准确性**：
   - 根据字幕时间戳确定
   - 确保覆盖完整的视觉演示过程

2. **类型判断**：
   - **PPT/板书**：静态内容，提取关键帧（2-5张）
   - **流程图/架构图**：静态内容，提取关键帧（1-3张）
   - **对比图/数据展示**：静态内容，提取关键帧（1-2张）
   - **效果演示**：动态过程，提取视频片段
   - **操作演示**：动态过程，提取视频片段

3. **插入位置**：
   - 明确指出应该插入到分析文档的哪个具体位置
   - 确保图片与文字内容紧密相关

4. **AI智能选帧指导**：
   - 如果PPT有多页，说明需要提取哪几页
   - 如果有逐步展示，说明选择完整版还是过程版
   - 如果有多个角度，说明选择哪个角度

### 示例

```markdown
## 视觉辅助素材清单

### 素材1：唯物主义三派系对比表
- **时间范围**：00:02:44 - 00:03:15
- **类型**：PPT/板书
- **内容说明**：展示朴素唯物主义、形而上学唯物主义、辩证唯物主义的对比表格，包括各自的观点和代表人物
- **插入位置**：核心洞察 → 核心概念 → 【唯物主义】定义之后
- **提取方式**：关键帧
- **帧数建议**：3-4张（PPT可能逐步展示三个派系）
- **选帧指导**：选择展示完整对比表的最后一张，如果有逐步展示，也保留中间关键步骤

### 素材2：物质定义的演进过程
- **时间范围**：00:05:14 - 00:05:50
- **类型**：流程图/示意图
- **内容说明**：展示"物质"定义从"具体实物"→"原子粒子"→"一切客观存在"的演进过程
- **插入位置**：逻辑链重构 → 证据链 → 证据2之后
- **提取方式**：关键帧
- **帧数建议**：2-3张（可能有箭头或时间线展示）
- **选帧指导**：选择展示完整演进过程的图，包含所有三个阶段

### 素材3：半唯物主义的困境演示
- **时间范围**：00:07:06 - 00:07:40
- **类型**：效果演示/动画
- **内容说明**：演示形而上学唯物主义如何在解释自然现象时成功，但在解释社会现象时失败，最终"投敌"到唯心主义
- **插入位置**：逻辑链重构 → 证据链 → 证据3之后
- **提取方式**：视频片段（30秒）或关键帧序列（5-6张）
- **帧数建议**：如果提取关键帧，需要5-6张展示完整过程
- **选帧指导**：选择展示"成功解释自然现象"→"无法解释社会现象"→"转向唯心主义"三个关键节点
```

### 注意事项

1. **不要过度标注**：
   - 只标注真正有视觉演示的部分
   - 如果只是讲话画面，不需要标注

2. **时间范围要精确**：
   - 根据字幕时间戳确定
   - 确保不遗漏关键内容

3. **说明要具体**：
   - 清楚说明视觉内容展示了什么
   - 解释为什么这个视觉内容重要

4. **插入位置要明确**：
   - 精确到具体的章节和段落
   - 确保图文对应
